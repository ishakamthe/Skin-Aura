# SkinAura

An AI-powered skincare ingredient analysis app built with React, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites
- Node.js (LTS version recommended)

### Installation

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- **React** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — styling
- **Framer Motion** — animations
- **React Router** — routing
- **Leaflet** — interactive AQI map

## Features

- Search skincare products by name or brand
- Safety & eco scores per product
- Ingredient-level hazard analysis
- AQI map with skin protection recommendations
- Filter by ingredients, scores, and brand
