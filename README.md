# 🌿 SkinAura

> Make smarter skincare choices — for your skin *and* the planet.

---

## 🚨 Problem

Consumers often lack clear and accessible information about the **environmental impact of skincare products**.
This leads to unintentional harm such as:

* 🌊 Water pollution
* 🧪 Toxic ingredient buildup
* 🌱 Ecological damage

---

## 💡 Solution

**SkinAura** analyzes skincare product ingredients and provides an **Eco Impact Score** to help users make **sustainable, informed decisions**.

---

## ✨ Features

* 🔍 Ingredient analysis for skincare products
* 🌱 Eco Impact Score (easy-to-understand rating)
* ⚠️ Identification of harmful or non-eco-friendly ingredients
* 📊 Clean and simple user interface
* 🧠 Promotes awareness of sustainable skincare

---

## 🛠️ Tech Stack

* **Frontend:** React + Vite
* **Backend:** Python
* **API:** Indian Government Open Data Platform (data.gov.in)

---

## 🚀 How It Works

1. User inputs a skincare product
2. SkinAura analyzes the ingredient list
3. Each ingredient is evaluated for environmental impact
4. A final **Eco Score** is generated
5. User gets insights + recommendations

---

## 🎯 Goal

To empower consumers to:

* Make eco-friendly skincare choices 🌍
* Reduce environmental damage 🌱
* Build awareness about sustainable products
